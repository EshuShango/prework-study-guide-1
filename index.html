HTML

  • HyperTextMarkupLanguage is the basic foundation a web/mobile app is built upon
  • Parent/child language is used often by devs -- ex: body element is the parent element which the child element header is nested within
  • The head element contains info about the webpage
  • The body element reps the visible content shown to the user

CSS

  • CascadingStyleSheets is utilized to "pretty up" the basic foundation of a web/mobile app
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element

Git

  • Git is a "version control" tool used by devs and non devs to help keep code organized
  • Git is a very useful tool for devs and should be "mastered"
  • git clone: retrieve an entire repo from a hosted location via URL
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is used to help a web applications interactive
  • JavaScript's main bread and butter are functions
  • JavaScript is one of the most poplular languages of the web
